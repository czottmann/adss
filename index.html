(Unofficial) Apple Developer System Status Feeds

Apple publishes two status pages for their various systems:

  • System Status show the statuses of most user-facing Apple services and stores.
  • Developer System Status displays developer-facing services.

Useful, but as a iOS/macOS developer I like my status updates both as RSS feeds and with a bit more urgency. So I've set up accounts on "my" social networks, and a feeds to relay the status updates as they happen. The feeds are updated every 2 minutes.

  • Mastodon: @AppleDeveloperSystemStatus@techhub.social
  • Bluesky: @AppleDeveloperSystemStatus.zottmann.dev
  • RSS feed: RSS

FYI: If you want to work with the raw data yourself, Apple publishes the updates as JSON and JSONP, respectively, at the following URLs:

  • System Status (JSON)
  • Developer System Status (JSONP)
